# Get-pip
Una herramienta hecha para obtener el pip de python3 y python2
# Instalacion
```
apt install python3 -y
apt install python2 -y
apt install git -y
cd $HOME
git clone https://github.com/Euronymou5/Get-pip
cd Get-pip
python3 pip.py
```
